# Contact-Management-System-cpp
A simple console-based Contact Management System built using C++. This project allows users to store and manage up to 50 contacts using basic Object-Oriented Programming concepts.
## Features
Add a new contact (Name, Phone Number, Email), 
View all saved contacts,
Search contact by name,
Delete contact using phone number,
Handles up to 50 contacts,
Proper memory management (no memory leaks)
